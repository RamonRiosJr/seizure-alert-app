# Security Policy

## Supported Versions

Use the latest version of the application for the most secure experience.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

**This application deals with health safety.** If you discover a security vulnerability, please treat it with urgency and discretion.

Please do **NOT** open a public issue.

Instead, please email the project maintainer directly at: **<security@coqui.cloud>** (or contact Ramon Rios Jr).

We will verify the issue and verify a fix promptly. A public disclosure date will be negotiated if needed.
